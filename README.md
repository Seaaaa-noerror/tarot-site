# 塔罗自学与日记复盘网站

这是一个用于塔罗学习、抽牌解读、日记复盘和资料检索的静态网页项目。它把塔罗牌学习、每日记录、素材库和文字索引整合在一起，适合用于个人塔罗自学与长期觉察记录。

## 主要功能

- 首页抽牌：输入问题后抽取塔罗牌，并生成对应牌面解读。
- 78 张牌学习：按大阿尔克纳和四组小阿尔克纳分类学习，每张牌有独立详情页和学习记录区。
- 日记记录回顾：记录当天抽到的牌和日记内容，并通过月历回看历史记录。
- 素材库：上传 EPUB、PDF、TXT、DOCX 等资料文件，自动提取可读文字并建立本地索引。
- 素材匹配：写日记时，网页会根据日记内容实时匹配素材库中的相关片段。
- 素材定位：点击匹配片段后，可跳转到素材内容定位页，高亮对应文字位置。

## 页面结构

- `index.html`：抽牌首页
- `detail.html`：抽牌结果详情页
- `study.html`：塔罗牌 78 张分类学习页
- `study-card.html`：单张牌学习详情页
- `journal.html`：日记记录与月历回顾页
- `journal-entry.html`：单日记录详情页
- `language.html`：塔罗语言学习素材库
- `material-reader.html`：素材内容定位页

## 本地使用

这是一个纯前端静态项目，不需要安装依赖。

直接用浏览器打开：

```text
index.html
```

也可以部署到 GitHub Pages、Netlify、Vercel 等静态网站托管平台。

## 数据存储说明

本项目的数据主要保存在当前浏览器本地：

- 日记记录保存在 `localStorage`
- 素材库文件和文字索引保存在 `IndexedDB`

因此，清理浏览器缓存、更换浏览器或更换设备，可能会导致本地数据不可见或丢失。


## 项目定位

这个网站不是占卜结论工具，而是一个帮助用户把抽牌、学习、复盘和资料阅读串联起来的自学系统。它更适合用于记录个人感受、整理牌义理解，以及长期观察自己的生活模式。
