# 考研学习监督台

这是一个 Windows 本地桌面小软件，用 PowerShell WinForms 实现，不需要安装依赖。

## 启动

- 双击桌面的“考研学习监督台”图标。
- 或双击项目里的 `start-learn-monitor.cmd`。

## 功能

- 记录学习时长、有效学习、计划完成度、正确率和每日进度。
- 数学一、408 已导入知识点按钮。
- 点击知识点按钮会按 `10%` 递增，超过 `100%` 后回到 `0%`。
- 数学一和 408 的预测分数由“节”权重计算，并考虑遗忘曲线、正确率和有效学习时长。
- 学习记录页使用树形关联控件：点击节名展开/收起细分知识点，双击节名记录整节学习，双击小知识点记录具体知识点学习。
- 真题研究页记录数学一、408、英语一 2000-2025 年真题是否做过和做了几遍，不保存真题内容。
- 英语一、政治当前按学习记录估算。
- 上岸概率由预测总分、目标分数线和已填写知识点覆盖度共同估算。

## 模型说明

数学一和 408 的每个知识点会先计算基础熟练度，再按距离上次复习的天数做遗忘衰减，并由最近 30 天的有效学习时长和正确率进行强化；每个节的当前有效熟练度由节内知识点汇总。目标院校概率会随着预测总分、目标分数线、学习记录数量和知识点覆盖度变化。

这个软件给的是学习监督和趋势预测，不是官方录取判断。
